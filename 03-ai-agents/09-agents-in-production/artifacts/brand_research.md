# Jornada de Dados — Brand Research (raw findings)

> Observação: a pesquisa pública disponível via web search não retornou diretamente um *manual de identidade visual* com paleta/hex/fontes. Abaixo estão os indícios textuais e as fontes encontradas; quando não foi possível obter HEX/typography/logo com precisão, uso recomendações seguras (e sinalizo).

## 1) Identidade/posicionamento (o que a marca diz)
- **Nome/Marca:** Jornada de Dados
- **Oferta/tema central:** trilhas de **Análise, Engenharia de Dados e Inteligência Artificial**; formação completa com **Python, SQL, Databricks e IA**.
- **Frase/claim (site):** “**20+ projetos práticos aplicáveis ao mercado** … **Comunidade ativa de 5.000+ profissionais**.” (suajornadadedados.com.br)
- **Frase (LinkedIn snippet):** “**Torne-se um Especialista em Dados, Python e SQL** … workshops e bootcamps … criar pipelines, ETLs e projetos … acesso vitalício.” (LinkedIn company overview)
- **Escopo (site/curso):** “Jornada de aprendizado” para **Engenharia de Dados + IA** e projetos práticos.

## 2) Tom de voz / marca (inferência a partir do texto encontrado)
- **Tom/voz:** profissional, orientado a resultados, com ênfase em prática (“projetos práticos”, “pipelines”, “ETLs”, “projetos reais”).
- **Posicionamento:** “trajeto”/“jornada” do **básico ao avançado**, com foco em empregabilidade/impacto em entregas (ex.: melhorar entregas e processos para analista/engenheiro/cientista de dados).
- **Público:** pessoas em Engenharia de Dados (e adjacentes como IA/Data Engineering/MLOps dependendo da trilha).

## 3) Typography hints
- Não foi encontrado, via search, um indicativo direto de fontes (ex.: “font-family”/Google Fonts) no conteúdo indexado.
- **Sugestão de família (inferida para marca tech em PT-BR):** uma sans moderna e limpa (ex.: **Inter / Manrope / IBM Plex Sans**) para UI e **uma grotesca/serif secundária** apenas se houver necessidade editorial.

## 4) Cores (hex)
- **Não foi possível extrair HEX confiável** de paleta oficial a partir dos resultados retornados.
- Portanto, por regra do fluxo (e para não travar entrega), este arquivo registra **que HEX não foi encontrado** e que será usado um palette padrão tech para a identidade visual.

## 5) Logo / uso (cues)
- A busca por “suajornadadedados.com.br logo” não retornou detalhes vetoriais/códigos da marca.
- Não foi encontrado um guia de uso (ex.: versões, área de proteção, tamanhos mínimos).

## 6) Tagline / slogans
- Não foi encontrado um *tagline* curto e explícito e indexado (ex.: “Do zero ao…” com frase exata), apenas claims longas sobre especialização e projetos.
